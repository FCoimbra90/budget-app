## Passo 2: Criar Repositório no GitHub

1. Após logar, clique no ícone de perfil (canto superior direito)
2. Clique em "Your repositories"
3. Clique no botão verde "+ New" (ou "Create repository")
4. Preencha assim:
   - **Repository name:** `budget-app`
   - **Description:** `Sistema de Controle Financeiro Pessoal`
   - **Public** (deixe público)
   - Deixe tudo mais desmarcar (não precisa de README, .gitignore já temos)
5. Clique em "Create repository"

---

## Passo 3: Fazer Push do Código (SEM TERMINAL)

Você tem duas opções:

### Opção A: GitHub Desktop (Mais Fácil - SEM TERMINAL)

1. Baixe GitHub Desktop: https://desktop.github.com/
2. Instale (duplo clique no .dmg)
3. Abra GitHub Desktop
4. Clique em "File" → "Clone Repository"
5. Cole o link do seu repositório (ex: `https://github.com/seu-usuario/budget-app`)
6. Escolha onde salvar (pode ser em qualquer lugar, não precisa ser em Google Drive)
7. Clique "Clone"
8. Abra a pasta do repositório clonado
9. Delete TODOS os arquivos lá (deixe vazio)
10. Copie os arquivos da pasta `budget-app` (aqui de local) para lá
11. GitHub Desktop vai detectar as mudanças automaticamente
12. Digite a mensagem: "Adicionar projeto Budget App"
13. Clique "Commit to master"
14. Clique "Push origin"
15. Pronto! Seu código está no GitHub

### Opção B: Upload Direto (Se Preferir)

Se não quiser instalar nada:

1. No GitHub, clique "Upload files" (dentro do seu repositório vazio)
2. Selecione TODOS os arquivos da pasta `budget-app`
3. Clique "Commit changes"
4. Pronto!

---

## Passo 4: Criar Conta no Railway

1. Abra https://railway.app/
2. Clique em "Start Free" ou "Sign up"
3. Clique em "Continue with GitHub"
4. Aprove a autorização
5. Pronto! Você está no Railway

---

## Passo 5: Fazer Deploy (SEM TERMINAL)

1. No Railway, clique em "Create New Project"
2. Clique em "Deploy from GitHub repo"
3. Selecione o repositório `budget-app`
4. Railway vai detectar que é um projeto Node.js automaticamente
5. Clique em "Deploy"
6. Aguarde (pode levar 2-3 minutos)
7. Quando terminar, Railway vai gerar uma URL tipo: `https://budget-app-xyz.railway.app`
8. **Clique no link e pronto!** Seu app está online! 🎉
