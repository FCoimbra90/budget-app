# 💰 Budget App — Como Usar

## ✅ O Sistema Está Pronto

Seu app de controle financeiro está instalado nesta pasta (`budget-app/`).

---

## 🚀 Como Abrir (SEM Terminal)

### Opção 1: Duplo Clique (Recomendado)

Na pasta `Orçamento Mensal`, você deve ver um arquivo chamado:
**▶ Iniciar Budget App.command**

Simplesmente **dê um duplo clique** e o sistema abre no navegador automaticamente.

(Na primeira vez, o Mac pode pedir uma confirmação de segurança — é normal)

---

### Opção 2: Se Preferir Usar o Terminal

Abra o Terminal e cole isto:

```bash
cd ~/GoogleDrive/Meu\ Drive/Finanças/Orçamento\ Mensal/budget-app
npm run dev
```

Depois abra o navegador em: **http://localhost:5173**

---

## 📖 O Que Fazer Agora

1. **Crie suas contas** — Banco, cartão de crédito, corretora
2. **Crie suas categorias** — Despesa, receita, com subcategorias
3. **Importe seus extratos** (próximo passo que vou implementar)
4. **Visualize no Dashboard** — Veja seu fluxo de caixa mensal
